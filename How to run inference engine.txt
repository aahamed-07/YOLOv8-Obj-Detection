========== HOW TO RUN INFERENCE ENGINE ==========

cd to the folder location
and in the terminal/cmd run the following script -

python Obj_detect.py

wait for few seconds to a minute and open/attach your webcam
, the inference engine will start and you will see person detection in realtime 
in a pop up window with direct video (Real Time) feed from your webcam.

to close the window and the program from running press Esc key.
